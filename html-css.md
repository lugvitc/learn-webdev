# Javascript

-   Created in 10 days!

## Basics

-   Variables: `let`, `const` and `var`
-   The `function` keyword:
    ```js
    function addOne(number) {
        return number + 1;
    }
    ```
-   Objects
-   Arrays
-   Arrow functions:

    ```
    const addOne = (number) => {
        return number + 1;
    };
    const addOne = (number) => number + 1;
    const addOne = number =>  number + 1;
    ```

-   functional programming - map, filter
